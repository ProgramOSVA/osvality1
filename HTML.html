¡¡APRENDE A PROGRAMAR EN CODIGO HTMl!!

HTML (HyperText Markup Language) es un lenguaje de programación que se utiliza para crear y estructurar páginas web y aplicaciones en línea. HTML se basa en etiquetas y elementos para definir el contenido y la estructura de una página web, como párrafos, listas, imágenes, enlaces, tablas, y otros recursos multimedia.

Algunas características de HTML son:

  • Es gratuito de codigo abierto
  • Se ejecuta en todos los navegadores
  • Es flexible e integra con otros lengaujes
  • Tiene un marcado limpio y consistente


ESTRUCTURA DE UNA PAGINA WEB

Cada documento HTML se puede dividir en dos partes principales :

1. El cuerpo: el cual contiene la parte del documento a ser mostrada por el browser.

2. El encabezado: el cual contiene informacion acerca del documento, pero el cual no se muestra.
